## Overview

This project implements a sales prediction system that analyzes historical sales data to forecast future sales trends. The model helps businesses make data-driven decisions for inventory management and sales strategy.

## Features

- Data preprocessing and cleaning
- Exploratory data analysis
- Multiple regression model implementation
- Model evaluation and comparison
- Sales forecasting capabilities

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Aditya7808/lava-sales-model-.git
cd lava-sales-model-
```

2. Create a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: .\venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```
